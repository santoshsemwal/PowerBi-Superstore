# 📊 Sales Dashboard using Power BI

An interactive and visually appealing Sales Analytics Dashboard created using Microsoft Power BI to analyze sales performance, profit trends, customer insights, and regional performance.

---

## 🚀 Project Overview

This dashboard helps visualize and understand:
- Total Sales Performance
- Profit Analysis
- Category-wise Sales
- Regional Insights
- Customer Trends
- Interactive Filtering & Data Exploration

The project demonstrates data visualization, dashboard designing, and business analytics skills using Power BI.

---

## 📌 Features

✅ Interactive Dashboard  
✅ KPI Cards for Quick Insights  
✅ Sales & Profit Analysis  
✅ Region-wise Performance Tracking  
✅ Category-wise Comparison  
✅ Dynamic Charts & Visualizations  
✅ Clean and User-Friendly UI  

---

## 🛠️ Tools & Technologies Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Excel Dataset

---

## 📈 Dashboard Visuals

The dashboard includes:
- Bar Charts
- Column Charts
- Pie/Donut Charts
- KPI Cards
- Ribbon Charts
- Filters & Slicers
